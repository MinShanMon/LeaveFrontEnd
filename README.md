# JavaSpring Email-OTP

This is login system with one time password which is send by email.

## create a java spring starter project with following setting:

- Type : Marven
- JavaVersion: 17
- Language: Java
- Spring Version: 2.7.6


#### dependencies
- spring web
- Thymeleaf
- Spring Boot Devtool
- JPA
- Spring Security
- Mail
- Mysql Connector
